## 🗺️ 学习路径导航

### 如果你从零开始：

1. **先看 roadmap.html** - 了解整体学习路线
2. **进入 index.html 第1章** - 开始入门学习
3. **依次完成 index.html 第2-6章** - 掌握基础和进阶
4. **进入 principles.html** - 学习底层原理
5. **最后 interview.html** - 准备面试

### 如果你已有基础：

- 直接跳到对应的章节查漏补缺
- 使用 interview.html 检测自己的掌握程度
